Introduction

Welcome to Introduction to Django 101.
This website contains notes of all classes as well as a live editor and a place to ask questions and doubts.

This course will equip you of the python programming language, which is one of the most lucrative and fun programming language to learn. It is a melodious blend of object oriented paradigms and some neat functional programming ideas. As far as web development in python is considered, Django is the recommended framework for beginners. Django is a popular framework, and very very actively maintained by the open source community behind it. The project’s claim to fame is to “take care of much of the hassle of web development.” The idea is that tons of functionality is already built into an out-of-the-box Django app when you get it fired up, such as database solutions and HTML templating abilities (i.e. make pages generate content on the fly), which enables faster prototyping. This is the reason many big startups use django (and continued to use django after scaling) like instagram, pinterest , prezi and disqus. Hence, its a very good time to learn django, and many new emerging startups are also hiring in this domain. The main topic which will be covered in the course are

    👉🏻 Intro to python, into to terminal, intro to git, setting up python and django
    👉🏻 Intro to Django , intro to heroku for deployment, into to regex
    👉🏻 Models and databases
    👉🏻 User authentication and forms
    👉🏻 Cookies and sessions
    👉🏻 Into to web dev with bootstrap and django templating
    👉🏻 Setting up django project on VPS (uwsgi and nginx/apache)
